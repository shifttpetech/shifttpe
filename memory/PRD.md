# ShiftPe – PRD (V1 Frontend)

**Tagline**: Swipe. Work. Earn.
**Type**: Two-sided gig marketplace (Students ⇄ Businesses)
**Stack**: Expo Router (SDK 54), React Native 0.81, Reanimated, Gesture Handler, AsyncStorage. Mocked data, no backend yet.

## Flows
Splash → Onboarding (3) → Role select → Phone login → OTP (mock, any 6 digits) → Profile setup → Home tabs.

## Student
Swipe (Tinder cards), Chats, Center FAB (urgent shifts), My Shifts (Applied/Accepted/Completed/Cancelled), Profile, Job Details, Match celebration, Chat thread, Notifications, Settings.

## Business
Dashboard (metrics + active jobs + applicants preview), Applicants (accept/reject), Center FAB (Create Shift), Chats, Profile.

## Design Tokens
Primary #FF3D57, Surface #FFFFFF, Text #0F172A. Poppins family, weights capped at 500. Radius pill / xxl 32 / xl 24. 8pt spacing grid.
